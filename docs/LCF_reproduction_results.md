# Paper B (LCF) — Reproduction Results

From-scratch implementation of "Content-free Logical Modification of LLM by Disentangling and Modifying Logic Representation" (AAAI 2025). Official repo had no code. Base model **Qwen/Qwen3-8B** (paper used Llama2/3/Vicuna/Mistral/ChatGLM3/Baichuan; we add Qwen3 per the project's model decision; paper models in progress).

## Setup actually run
- LFUD scenario split 540/60/204 (train/val/test); 540 valid conclusions generated by **GPT-3.5-turbo** (paper-faithful), 540 invalid = original LFUD fallacious sentences.
- LCF module trained on Qwen3-8B hidden reps (layers 10–30, attn+MLP), AdamW 1e-3, 10 epochs. Distinctiveness-selected sub-layers (top-10) are **attn 10–19 + 27, mlp 14** — matches the paper's "attention 10–20 helps most, 15–20 best" observation.
- Eval: Conclusion Generation (η=0.5) judged by **GPT-4o** + a trained distilbert validity discriminator (eval-set accuracy 1.0); Fallacy Identification 4-option (η=4.5) via option log-probs → Accuracy, ΔProb.

## Results (Qwen3-8B, 204-item test)

| Variant | ValidGPT4↑ | ValidTrained↑ | PPL↓ | Acc↑ | ΔProb↑ |
|---|---|---|---|---|---|
| original | 47.06 | 82.35 | 3.80 | 31.86 | 3.96 |
| **+LCF** | 47.06 | 76.47 | **2.02** | 31.37 | **7.83** |

## Findings (honest)

1. **Fallacy Identification — LCF works (headline).** ΔProb **3.96 → 7.83 (≈2×)**: LCF shifts logic representations so the model assigns higher probability to the valid option over invalid ones. Same **direction and comparable magnitude** as the paper (Llama2 ΔProb −1.89 → 6.29). This is the paper's central claim and it reproduces. Accuracy is ~flat (31.9 → 31.4), as in the paper where ΔProb moves more than Acc.

2. **Conclusion Generation — muted on Qwen3 + greedy.** With clean (first-line) conclusions, +LCF and original produce ~identical text → ValidGPT4 tied at 47.06. Reason: generation uses the gentle **η=0.5** nudge, which does not change greedy argmax tokens on Qwen3-8B. The paper saw generation gains on Llama2; the effect is decoding/model-dependent. (An earlier 41→47 gap was an artifact of judging the models' post-conclusion rambling, removed by first-line cleaning.)

3. **PPL drops (3.80 → 2.02)** under LCF — the modified hidden states yield lower perplexity on the produced conclusions.

4. **Trained-discriminator caveat.** The distilbert discriminator hits 1.0 accuracy on held-out LFUD scenarios but is sensitive to the generation distribution (initial deberta-v3 runs were degenerate due to a broken SentencePiece tokenizer under transformers 5.x; switched to distilbert fast tokenizer). Treat **ValidGPT4 as the primary validity metric** (paper's main judge), ValidTrained as secondary.

## Layer-distinctiveness (matches paper)
Top selected sub-layers by valid/invalid separability: attn L12(.86) L17(.86) L15(.83) L19(.82) L13(.81) L10(.80) L27(.77) L16(.77) L11(.77), mlp L14(.77). Concentration in attention layers 10–19 reproduces the paper's modification-layer analysis.

## Why Qwen3 first, then Llama-2 (model choice note)
Qwen3-8B was used first because it was already cached locally → fastest end-to-end validation of the from-scratch LCF. The paper's models are: **Llama-2-7b-chat-hf** (headline), Llama-3.1-8B-Instruct, vicuna-7b-v1.5, Mistral-7B-Instruct-v0.2, chatglm3-6b, Baichuan2-7B-Chat. Llama-2-7b-chat IS accessible with our HF token (Llama-3.1 is still gated), so it is now being run for a faithful, paper-comparable result; the Qwen3 result additionally shows LCF generalizes beyond the paper's models.
